# CustomMealRewards_GoogleForm
利用github提供的靜態網頁加上Google Form當後端

前端:static web  
後端:Google Form  
資料庫:Google Sheets  

1.填寫美化的報名表單  
2.自動歸入Excel試算表  
3.只要同意發送電子報就會發送信件至填寫的Email  
4.有人發送表單就會跳出LINE通知給老闆
